HTML

Es una de las partes más básicas de todas las páginas web, por lo que es muy importante aprenderlo si se quiere tener una carrera en el mundo del desarrollo web.

CSS

(CSS) es un lenguaje de programación que sirve para determinar el diseño de los documentos electrónicos. Con la ayuda de unas sencillas instrucciones -presentadas en forma de código fuente claro, los elementos del sitio web, como el diseño, el color y la tipografía, pueden adaptarse como se desee.

PYTHON
